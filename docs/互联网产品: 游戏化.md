# 游戏让人着迷

玩家们在现实中或许只能偶尔体会到这样的愉悦，但在玩自己喜欢的游戏时，他们几乎时时刻刻都能体会到。

- 在现实世界中，要到哪里才能找到像游戏里那样能完全活出自我、时时刻刻保持专注和投入的地方？
- 要到哪里才能找到游戏里的力量感、英勇无故的目标感和团结一心的感觉？
- 要到哪里才能找到游戏里那种有了成就之后的振奋和创造悸动？
- 要到哪里才能找到游戏里在成功和团队获胜后心跳加速的快感？
